Inicializar package.json: npm init --y
2. Instalar Express: npm i express
2.1 Inslatar types de express: npm i -D @types/express
3. Instalar Nodemon para reiniciar el servidor y ts-node (TypeScript de CDLI de Node), e instalar TypeScript: npm i -D nodemon ts-node typescript
4. Instalar Colors para los mensajes del servidor: npm i colors
5. Instalar DOTENV: npm install dotenv
6. Instalar Mongoose para las conexiones con mongodb: npm install express mongoose
7. Instalar validaciones de express en los endpoints: npm i express-validator
8. Instalar CORS: npm i cors | types: npm i -D @types/cors
9 Instalar Morgan: npm i morgan | npm i -D @types/morgan
10 Instalar bcrypt: npm i bcrypt | npm i -D @types/bcrypt
11 Instalar nodemailer para envio de correos npm i nodemailer | npm i -D @types/nodemailer
